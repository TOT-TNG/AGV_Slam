Cấu hình Node

Nhấp chọn một node trên bản đồ để gán tên chức năng, loại vị trí và hành động mặc định.

Bạn có thể gán mỗi node thành một vị trí nghiệp vụ cụ thể, ví dụ: Conveyor01, ChargeStation01, DropZoneA, HomeBase, BufferZone, ParkingSpot01…
Chưa gán chức năng
Conveyor - Băng tải / Điểm nhận/gửi hàng
Charger - Trạm sạc tự động
Dropoff - Điểm thả hàng / Giao hàng
Home - Điểm về nhà / Vị trí gốc
Buffer - Khu vực chờ / Khu vực đệm
Parking - Khu vực đỗ xe / Đợi
